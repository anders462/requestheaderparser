 # Request Header Parser Microservice:

 API Uses: Node.js/Express.js

 1. User Story: I can get the IP address, language and operating system for my browser.

 Example result:
 {
"ipaddress": "107.205.9.222",
"language": "en-US",
"software": "Mac OS X 10.11.1"
}
